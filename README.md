# CodeAlpha_task3_obj_tracking
## Object Detection and Tracking using YOLOv8

## Project Overview
This project performs real-time object detection and tracking on videos using the YOLOv8 model. It detects multiple objects, assigns unique tracking IDs, and follows their movement throughout the video.

## Features
- Real-time object detection
- Multi-object tracking
- Unique tracking IDs for detected objects
- Supports video file input
- High-speed inference using YOLOv8

## Technologies Used
- Python
- YOLOv8 (Ultralytics)
- OpenCV
- Google Colab

## Project Structure

CodeAlpha_Task3/
   object_detection_tracking.ipynb
   input_video.mp4
   output_video.avi
   README.md


## Installation

Install the required libraries:

```bash
pip install ultralytics opencv-python
```

## How to Run

1. Open the notebook in Google Colab.
2. Install the required libraries.
3. Upload a video file.
4. Run all cells.
5. The model detects and tracks objects.
6. The processed output video is saved automatically.

## Output
The output video contains:
- Bounding boxes around detected objects
- Object class names
- Confidence scores
- Unique tracking IDs
